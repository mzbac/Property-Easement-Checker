Property Easement Checker

Use this tool to check if a property you're considering has easement issues.

How to Use This Tool

  1. Upload two images: a satellite image and an easement image.
  2. Match features on both images:
    • Find a recognizable spot (like a corner or intersection) on the satellite image.
    • Click on that spot.
    • Find the same spot on the easement image and click it there too.
    • Repeat this process at least 3 times with different spots.
  3. If you make a mistake, use the "Clear Points" buttons to start over.
  4. Once you've matched at least 3 spots, click the "Process Images" button.
  5. The tool will combine the images, showing the easement overlay on the satellite image.
